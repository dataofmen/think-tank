# 사용하는 도구 목록 (Tools)

## 업무 도구 (work-manager 연동)

### 리서치 도구
- **Condense**: 정성 리서치 분석 도구
- **Qualtrics**: 설문 플랫폼 (웹/인앱)
- **Jira**: 과제 관리 (UXRESEARCH 프로젝트)
- **Slack**: 팀 커뮤니케이션

### 데이터 도구
- Google Sheets: 데이터 분석, 차트
- Confluence: 회의록, 리서치 결과 공유
