# Data Analysis

## Core Ideas

- [[Fixed position sizing structurally misprices risk across assets with different volatilities]] — 고정 사이즈 포지셔닝의 구조적 결함: 변동성 차이를 무시하면 리스크가 8배 차이남
- [[Volatility targeting normalizes risk exposure through the ratio of target to realized volatility]] — 변동성 타겟팅의 핵심 공식과 실무 가드레일
- [[EWMA with decay factor 0.94 is a widely adopted standard for short-term volatility estimation]] — RiskMetrics 표준 변동성 추정법, 0.94 감쇠 계수의 근거와 한계
- [[Volatility targeting improves risk-adjusted returns by reducing exposure during high-volatility regimes]] — 백테스트 기반 증거: Sharpe ratio 개선과 whipsaw 한계
- [[Correlated contracts on the same event represent one position regardless of nominal diversification]] — 상관관계 무시한 분산투자는 집중 리스크를 은폐함
