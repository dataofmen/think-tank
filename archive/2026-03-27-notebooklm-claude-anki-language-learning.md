# NotebookLM + Claude + Anki: How to Learn Any Language by Watching Series

### 핵심 시스템
NotebookLM, Claude, Anki 3가지 도구를 결합한 언어 학습 시스템:
- **NotebookLM**: TV 시리즈 대본/자막에서 학습 소재 추출
- **Claude**: 추출된 소재를 Anki 플래시카드 형식으로 포맷팅
- **Anki (FSRS)**: 간격 반복 스케줄링으로 장기 기억 정착

### 주요 섹션
1. **망각 곡선 원리** (Forgetting Curve Principles)
2. **추출 파이프라인** (Extraction Pipeline): NotebookLM → Claude → Anki
3. **Anki FSRS 설정**: 목표 retention 0.90
4. **추천 워크플로우**: 카드 학습 후 에피소드 시청

### 방법론 특징
- TV 시리즈 대본을 통한 맥락적 학습 (contextual learning)
- 교과서 대신 실제 대화와 감정적 앵커링(emotional anchoring) 활용
- 전통적 학습앱 대비 실제 대화 맥락에서의 retention 향상 주장

### 결과
- 시리즈 1편에서 504장의 플래시카드 생성
- 교과서 제로, AI 2개 + 알고리즘 1개로 완성
