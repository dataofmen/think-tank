> "Models can code, write, design, and plan. What they can't do is share your taste."

## Metrics

- 30,000 GitHub stars in 3 weeks

## Core Concept

Paperclip positions itself as the layer between "your ideas" and "AI agent execution". The key insight is that while AI models have become increasingly capable at execution tasks, they lack the organizational context and values alignment that human teams develop over time.

## Key Features

1. **Heartbeat System**: Maintains agent identity, goals, and quality standards across sessions
2. **Team Structure**: Define roles and responsibilities for multiple agents
3. **Routines**: Automate recurring workflows
4. **Skills Repository**: Shareable agent capabilities via skills.sh

## Use Cases (Early Adopters)

- Security firms running automated audits
- Marketing agencies integrating AI alongside human teams
- Roofing company using satellite imagery analysis for lead generation

## Philosophical Position

"AI can do anything, but it can't share your values" — this is Paperclip's core philosophy. The orchestration layer encodes organizational values and context so agents can operate autonomously while maintaining alignment with business objectives.
