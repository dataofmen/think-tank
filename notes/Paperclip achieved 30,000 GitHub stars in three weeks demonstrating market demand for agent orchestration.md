Paperclip은 2026년 3월 출시 후 3주 만에 GitHub 스타 30,000개를 달성했다. 이는 AI 에이전트 오케스트레이션 분야에서 가장 빠른 성장 중 하나다.

빠른 채택의 이유:
1. 오픈소스 — 기업이 자체 호스팅 가능
2. 벤더 중립 — 특정 모델(Claude, GPT 등)에 종속되지 않음
3. 실제 문제 해결 — 다수 에이전트 동시 실행 시 워크플로우 혼란을 해결

이 성장 속도는 에이전트 오케스트레이션이 실제 사용자 니즈라는 시장 검증이다.
