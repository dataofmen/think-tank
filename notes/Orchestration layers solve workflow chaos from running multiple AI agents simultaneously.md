다수의 AI 에이전트를 동시에 실행하면 워크플로우 혼란이 발생한다: 어떤 에이전트가 어떤 작업을 하는지, 결과를 어떻게 통합할지, 에이전트 간 의존성을 어떻게 관리할지.

오케스트레이션 레이어(Paperclip 등)는 이 문제를 해결한다:
- 아이디어 입력 → 오케스트레이션 → 에이전트 팀 실행 → 결과 통합
- Heartbeat 시스템으로 에이전트 정체성, 목표, 품질 기준 유지
- 루틴으로 반복 작업 자동화

이는 단일 에이전트 워크플로우에서 멀티 에이전트 워크플로우로 확장할 때 필수적인 레이어다.
